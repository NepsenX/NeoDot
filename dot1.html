About

NeoDot is a lightweight, web-based chatbot built using HTML, CSS, and JavaScript. It is designed to be simple and easy to customize, making it ideal for static websites or embedded widgets.

How It Works

NeoDot responds to user inputs using predefined logic and rules coded in JavaScript. It does not use any AI or machine learning by default. Instead, it offers a straightforward, rule-based chat experience that you can tailor to your needs.

AI Integration

NeoDot is not an AI model but can connect to services like ChatGPT to generate dynamic responses and Mermaid.js diagrams.
